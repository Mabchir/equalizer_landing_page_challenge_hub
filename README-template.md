### What I learned

- how to position a background image

```css
body {
  background-image: url("./assets/bg-main-desktop.png");
  background-repeat: no-repeat;
  background-position: 0% 20%;
  background-size: cover;
  min-height: 100vh;
}
```

- how to make an image overflow

```css
.container {
  overflow: hidden;
}

.img {
  position: relative;
  bottom: 40px;
}
```
